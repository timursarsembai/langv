# FFmpeg для LangV Player

Эта папка должна содержать `ffmpeg.exe` для генерации миниатюр на таймлайне.

## Как добавить FFmpeg:

1. Скачайте FFmpeg с официального сайта: https://ffmpeg.org/download.html
   - Или используйте готовые сборки: https://www.gyan.dev/ffmpeg/builds/
   - Рекомендуется: `ffmpeg-release-essentials.zip` (~80 MB)

2. Распакуйте архив и скопируйте `ffmpeg.exe` в эту папку:
   ```
   LangVPlayer/
   └── ffmpeg/
       ├── ffmpeg.exe    ← Нужен этот файл
       └── README.md
   ```

3. При сборке проекта `ffmpeg.exe` будет автоматически скопирован в папку с приложением.

## Минимальные требования:

- Нужен только `ffmpeg.exe` (не нужны `ffplay.exe` и `ffprobe.exe`)
- Размер: ~80-120 MB

## Если FFmpeg отсутствует:

Плеер будет работать полностью, но при наведении на таймлайн будет показываться только время без миниатюры кадра.

## Лицензия FFmpeg:

FFmpeg распространяется под лицензиями LGPL/GPL. При распространении приложения ознакомьтесь с условиями: https://ffmpeg.org/legal.html
